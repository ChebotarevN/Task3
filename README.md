# Task3
Наблюдатель
----------------
**Описание**:  Это приложение позволяет запускать таймер, который через оповещения перезапускает анимацию каждые 20 секунд, и запускает видео спустя 5 секунд.
 - **Технологический стек**: JavaFX.
 - **Статус**:  1.0.
